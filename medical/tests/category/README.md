# Медицинский тренажер - Руководство

## 📋 Что это?

Веб-приложение для тренировки знаний по медицинским тестам. Работает полностью на клиентской стороне, не требует сервера или базы данных.

## 🚀 Установка на GitHub Pages

1. **Создайте структуру папок в вашем репозитории:**
   ```
   medical/
   └── tests/
       └── category/
           ├── index.html
           ├── style.css
           ├── app.js
           └── questions.txt
   ```

2. **Скопируйте файлы** в соответствующую папку

3. **Включите GitHub Pages:**
   - Зайдите в Settings → Pages
   - Source: Deploy from a branch
   - Branch: main (или master) → /root
   - Сохраните

4. **Готово!** Приложение доступно по адресу:
   `https://cosmocalamary.github.io/medical/tests/category/`

## 📝 Формат файла questions.txt

### Базовая структура:

```
Q: Текст вопроса?
A: Первый вариант ответа
B: Второй вариант ответа
C: Третий вариант ответа
D: Четвертый вариант ответа
CORRECT: A

[пустая строка между вопросами]

Q: Следующий вопрос?
...
```

### Правила:

1. **Вопрос** начинается с `Q:`
2. **Варианты ответов** с `A:`, `B:`, `C:`, `D:`, `E:` и т.д.
3. **Правильные ответы** с `CORRECT:`
   - Один правильный: `CORRECT: A`
   - Несколько правильных: `CORRECT: A,C,D`
4. **Между вопросами** обязательно пустая строка

### Примеры:

**Один правильный ответ:**
```
Q: Какой орган вырабатывает инсулин?
A: Печень
B: Поджелудочная железа
C: Щитовидная железа
CORRECT: B
```

**Несколько правильных ответов:**
```
Q: Признаки инфаркта (может быть несколько)?
A: Боль в груди
B: Высокая температура
C: Одышка
D: Холодный пот
CORRECT: A,C,D
```

## 🔄 Как добавить свои вопросы

### Вариант 1: Прямое редактирование (для небольшого количества)

Просто отредактируйте `questions.txt` в GitHub веб-интерфейсе или локально.

### Вариант 2: Массовое добавление

Если у вас вопросы в другом формате (Excel, Word, текст), вы можете:

1. **Структурируйте их в следующем виде** (например, в Excel):
   - Колонка A: Вопрос
   - Колонка B: Вариант A
   - Колонка C: Вариант B
   - Колонка D: Вариант C
   - Колонка E: Вариант D
   - Колонка F: Правильные ответы (A или A,C,D)

2. **Используйте формулу для конвертации** (в Excel):
   ```
   ="Q: "&A2&CHAR(10)&"A: "&B2&CHAR(10)&"B: "&C2&CHAR(10)&"C: "&D2&CHAR(10)&"D: "&E2&CHAR(10)&"CORRECT: "&F2&CHAR(10)
   ```

3. **Или используйте Python скрипт** (сохраните как `convert.py`):
   ```python
   # Пример: если ваши вопросы в CSV
   import csv
   
   with open('questions.csv', 'r', encoding='utf-8') as csvfile:
       reader = csv.DictReader(csvfile)
       with open('questions.txt', 'w', encoding='utf-8') as output:
           for row in reader:
               output.write(f"Q: {row['Вопрос']}\n")
               output.write(f"A: {row['Вариант_A']}\n")
               output.write(f"B: {row['Вариант_B']}\n")
               output.write(f"C: {row['Вариант_C']}\n")
               if row.get('Вариант_D'):
                   output.write(f"D: {row['Вариант_D']}\n")
               output.write(f"CORRECT: {row['Правильный']}\n\n")
   ```

4. **Запустите:** `python convert.py`

## ✨ Функции приложения

### Основные возможности:

✅ **Умная система повторений** - вопросы, на которые вы ответили неправильно, будут появляться чаще

✅ **Выбор размера серии** - 10/20/30/50 вопросов

✅ **Статистика:**
- Общая статистика по всем вопросам
- Статистика текущей сессии
- Детальная статистика по каждому вопросу
- Список самых сложных вопросов

✅ **Адаптивный дизайн** - работает на iPhone, Android планшете, ПК

✅ **Светлая/темная тема** - переключается кнопкой в правом нижнем углу

✅ **Сохранение прогресса** - все данные хранятся локально в браузере

✅ **Клавиатурные shortcuts:**
- Enter - проверить/следующий вопрос
- 1-9 - выбрать вариант ответа

### Как это работает:

1. **Выберите количество вопросов** на стартовом экране
2. **Отвечайте на вопросы** - выбирайте один или несколько вариантов
3. **Проверьте ответ** - кнопка "Проверить"
4. **Смотрите результат:**
   - ✅ Зеленый = правильно
   - ❌ Красный = неправильно (покажет правильный ответ)
5. **Следующий вопрос** или завершите серию
6. **Просмотрите результаты** серии
7. **Анализируйте статистику** - найдите слабые места

## 🎯 Алгоритм умного повтора

Каждому вопросу присваивается "вес" в зависимости от вашей успеваемости:

- **Новый вопрос** = вес 1 (базовый)
- **Правильный ответ** = вес уменьшается
- **Неправильный ответ** = вес увеличивается

Вопросы с большим весом выпадают чаще. Это помогает сфокусироваться на сложных темах.

## 💾 Хранение данных

Все данные хранятся в **localStorage** браузера:

- `medicalTestProgress` - статистика по вопросам
- `theme` - выбранная тема

**Важно:** Данные привязаны к браузеру. Если вы используете разные устройства или браузеры, прогресс будет отдельный.

### Сброс прогресса:

Нажмите "Сбросить прогресс" на главном экране.

## 🔧 Кастомизация

### Изменить цвета:

Откройте `style.css` и измените переменные в `:root`:

```css
:root {
    --primary: #3b82f6;      /* Основной цвет */
    --correct: #10b981;      /* Правильный ответ */
    --incorrect: #ef4444;    /* Неправильный ответ */
}
```

### Изменить размеры серий:

Откройте `index.html` и измените значения `data-count`:

```html
<button class="series-btn" data-count="10">10 вопросов</button>
<button class="series-btn" data-count="25">25 вопросов</button>
```

Создайте дополнительные папки:

        └── ...

## 🐛 Устранение неполадок

### Вопросы не загружаются:

1. Проверьте, что `questions.txt` находится в той же папке, что и `index.html`
2. Проверьте формат файла - обязательно UTF-8
3. Откройте консоль браузера (F12) и посмотрите ошибки

### Прогресс не сохраняется:

1. Проверьте, что браузер позволяет использовать localStorage
2. Не используете ли вы режим инкогнито?
3. Проверьте настройки конфиденциальности браузера

### Не работает на мобильном:

1. Очистите кэш браузера
2. Обновите страницу
3. Проверьте подключение к интернету (для первой загрузки)
